Using the atlas index to find places on a map

How would one find where Mt Tahat is using an atlas?

You need to use the atlas index. Under the letter “T” in the index you will find: Tahat, Mont mtn Algeria 51 D7 23°18'N 5°37'E

  • The name of the feature (listed in alphabetical order)
  • The type of feature, for example river, mountain
  • The country
  • The page number in the atlas (usually the largest scale)
  • The alphanumeric grid reference
  • The latitude and longitude co-ordinates
