Conventional signs on atlas maps

Conventional signs are the symbols used to represent features on maps. A standardised set of symbols is used on the maps throughout this atlas. Each symbol takes the form of a point (town, domestic airport), line (railway, river) or area/polygon (dam, mountain range).

These symbols are contained in a key attached to each map. The key contains only the symbols relevant to its particular map.

The colours of the symbols are standardised and the symbols are grouped according to:

  • Relief and physical features
  • Water features
  • Communications
  • Administration boundaries
  • Settlement
  • Tourism
