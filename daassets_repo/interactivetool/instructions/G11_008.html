Converting vector data into a digital format

Digitising is the process of converting geographic features on a paper map into a digital format using a digitising tablet. The data is captured using a mouse or a digitiser linked to a computer. Vector data is captured in a very similar way to this task. Practise this skill yourself using the topographic map of Barberton.

GIS digitising task

Use the drawing tool to ‘digitise’ (trace) the following vector layers off the topographic map of Barberton.

  • In black, capture all the trig. beacons (point data) by drawing a dot on them.
  • In red, capture the R38 and R40 arterial route (line data) by tracing the road.
  • In green, capture the Barberton nature reserve (polygon data) by tracing the boundary and colouring it in.
