Working with vector data

Vector data

Vector data is made up of three basic data types: points, lines and polygons.

Point Line Polygon
A point feature is displayed at one particular (x, y) location (eg. information received from a GPS). A line consists of a series of (x, y) co-ordinates joined together, with each having the same (z) feature (eg. a road or river). A polygon consists of a series of (x, y) co-ordinates that join together to complete a boundary (eg. a vegetation type or land use zone).
