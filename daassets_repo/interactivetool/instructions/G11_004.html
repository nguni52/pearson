Working with raster data

Raster data

Satellite and airborne sensors generally generate raster data, consisting of pixels that store measured values of a phenomenon (usually surface reflectance) as it varies across the Earthâ€™s surface. Raster data generated this way is like a picture taken with a digital camera.

Raster data can also represent any phenomenon or value in its pixels. A very common and very useful application of raster data is in DEMs or Digital Elevation Models, in which each pixel represents elevation.
